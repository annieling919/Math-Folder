## 📚 知识覆盖

### 题型 1：判断事件类型
- 独立事件：放回 = 独立；不放回但未知前结果 = 等效独立
- 相依事件：不放回且已知前结果 = 相依

### 题型 2：独立事件概率计算
- 公式：P(A 且 B) = P(A) × P(B)
- 示例：抛硬币正面 + 掷骰子6点：P = 1/2 × 1/6 = 1/12

### 题型 3：相依事件概率计算
- 公式：P(A 且 B) = P(A) × P(B|A)
- 示例：10球(3红7绿)，不放回先红后红：P = 3/10 × 2/9 = 1/15

### 补充题型：不放回 + 未知前结果
- 公式：P(后事件) = Σ(各前情况概率 × 对应后概率)

## 🎮 交互模式

1. **批量练习模式** - 一次出多道题，逐题判题分析
2. **单题练习模式** - 单独出题练习
3. **讲解模式** - 概念讲解
4. **复习模式** - 知识点复习
